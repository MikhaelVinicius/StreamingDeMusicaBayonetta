# Descrição

#### Streaming de música 100% funcional, projetado em React,Tailwind, Redux e RapidAPI.

# Instalação

#### 1- Certifique-se que versões recentes do node, npm e yarn estão instalados em sua máquina.
#### 2- Importe este repositório e abra preferencialmente em seu VSCode.
#### 3- Abra o seu navegador padrão
#### 4- Abra o terminal do seu vs code e digite "yarn dev" ou "npm run dev"
#### 5- Copie e cole o link exposto no seu navegador (Geralmente: "http://localhost:3000/")
